# 初始化
## 前端
使用 Next.js 官方模板 + Shadcn 初始化
```bash
cd frontend
pnpm create next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
pnpm dlx shadcn@latest init
```


ps.使用npm方式
```bash
cd frontend
npx create-next-app@latest --typescript --app --src-dir --eslint --tailwind --import-alias "@/*"
npx shadcn@latest init
```
